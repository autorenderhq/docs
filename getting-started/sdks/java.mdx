## Installation

<CodeGroup>

```kotlin Gradle
implementation("io.autorender:autorender-java:0.0.1")
```

```xml Maven
<dependency>
  <groupId>io.autorender</groupId>
  <artifactId>autorender-java</artifactId>
  <version>0.0.1</version>
</dependency>
```

</CodeGroup>

## Authentication

Generate an API key from your AutoRender dashboard and set it as an environment variable:

```bash
export AUTORENDER_API_KEY=your-api-key
```

## Usage

```java
import io.autorender.client.AutorenderClient;
import io.autorender.client.okhttp.AutorenderOkHttpClient;

// Reads AUTORENDER_API_KEY from the environment automatically
AutorenderClient client = AutorenderOkHttpClient.fromEnv();
```

Or pass the key explicitly:

```java
AutorenderClient client = AutorenderOkHttpClient.builder()
    .apiKey("your-api-key")
    .build();
```

## Upload a File

```java
import io.autorender.models.uploads.UploadCreateParams;
import io.autorender.models.uploads.UploadCreateResponse;
import java.nio.file.Paths;

UploadCreateParams params = UploadCreateParams.builder()
    .fileName("photo.jpg")
    .file(Paths.get("photo.jpg"))
    .build();

UploadCreateResponse upload = client.uploads().create(params);
System.out.println(upload.fileNo());  // e.g. "fi_abc123"
System.out.println(upload.url());     // CDN URL
```

## Upload from URL

```java
import io.autorender.models.uploads.UploadCreateFromUrlParams;

UploadCreateFromUrlParams params = UploadCreateFromUrlParams.builder()
    .remoteUrl("https://example.com/image.jpg")
    .build();

client.uploads().createFromUrl(params);
```

## List Files

```java
import io.autorender.models.files.FileListParams;
import io.autorender.models.files.FileListResponse;

FileListParams params = FileListParams.builder()
    .limit(10L)
    .build();

FileListResponse result = client.files().list(params);
result.files().forEach(file ->
    System.out.println(file.fileNo() + " " + file.name() + " " + file.url())
);
```

## Get File Details

```java
import io.autorender.models.files.FileRetrieveResponse;

FileRetrieveResponse file = client.files().retrieve("fi_abc123");
System.out.println(file.data().url());
```

## Rename a File

```java
import io.autorender.models.files.FileRenameParams;

FileRenameParams params = FileRenameParams.builder()
    .fileNo("fi_abc123")
    .name("new-name")
    .build();

client.files().rename(params);
```

## Delete a File

```java
client.files().delete("fi_abc123");
```

## Folders

```java
import io.autorender.models.folders.FolderCreateParams;
import io.autorender.models.folders.FolderRenameParams;

// List folders
client.folders().list();

// Create a folder
FolderCreateParams createParams = FolderCreateParams.builder()
    .name("products")
    .build();
var folder = client.folders().create(createParams);

// Rename a folder
FolderRenameParams renameParams = FolderRenameParams.builder()
    .folderNo(folder.data().folderNo())
    .name("product-images")
    .build();
client.folders().rename(renameParams);

// Delete a folder
client.folders().delete(folder.data().folderNo());
```

## Multipart Upload

For large files (>100MB), use multipart upload to split the file into parts:

```java
import io.autorender.models.multipartuploads.MultipartUploadStartParams;
import io.autorender.models.multipartuploads.MultipartUploadCompleteParams;
import java.net.http.HttpClient;
import java.net.http.HttpRequest;
import java.net.http.HttpResponse;
import java.net.URI;
import java.nio.file.Files;
import java.nio.file.Paths;

byte[] fileData = Files.readAllBytes(Paths.get("large-video.mp4"));

// Start session
MultipartUploadStartParams startParams = MultipartUploadStartParams.builder()
    .fileName("large-video.mp4")
    .fileSize((long) fileData.length)
    .build();
var session = client.multipartUploads().start(startParams);

// Upload parts
HttpClient http = HttpClient.newHttpClient();
HttpRequest req = HttpRequest.newBuilder()
    .uri(URI.create(session.parts().get(0).url()))
    .PUT(HttpRequest.BodyPublishers.ofByteArray(fileData))
    .build();
HttpResponse<Void> res = http.send(req, HttpResponse.BodyHandlers.discarding());
String etag = res.headers().firstValue("etag").orElseThrow();

// Complete
MultipartUploadCompleteParams completeParams = MultipartUploadCompleteParams.builder()
    .uploadId(session.uploadId())
    .addPart(MultipartUploadCompleteParams.Part.builder()
        .partNumber(1L)
        .etag(etag)
        .build())
    .build();
client.multipartUploads().complete(completeParams);
```

## Error Handling

```java
import io.autorender.errors.AutorenderServiceException;
import io.autorender.errors.NotFoundException;

try {
    client.files().retrieve("fi_notexist");
} catch (NotFoundException e) {
    System.out.println(e.statusCode());  // 404
    System.out.println(e.getMessage());
} catch (AutorenderServiceException e) {
    System.out.println(e.statusCode());
    System.out.println(e.getMessage());
}
```
