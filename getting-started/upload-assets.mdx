## Endpoint

```bash
POST https://upload.autorender.io/api/v1/uploads
Authorization: Bearer YOUR_API_KEY
Content-Type: multipart/form-data
```

## Minimum request

```bash
curl -X POST \
  https://upload.autorender.io/api/v1/uploads \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -F "file=@./shoe.jpg" \
  -F "file_name=shoe.jpg"
```

## Recommended fields

| Field | Why use it |
| --- | --- |
| `folder` | Keep assets grouped by feature/domain (`products`, `campaigns`) |
| `tags` | Improve filtering and admin workflows |
| `custom_id` | Link files back to your internal model IDs |
| `metadata` | Persist business context (merchant, sku, locale) |

## Response you should store

- `file_no`
- `path`
- `url`

These values are enough to track assets internally and generate transformation URLs later.
