## URL anatomy

```text
https://assets.autorender.io/{workspace}/{transformations}/{asset_path}
```

| Part | Example | Purpose |
| --- | --- | --- |
| Workspace | `WORKSPACE123` | Targets your project |
| Transformations | `w_1200,h_630,c_fill` | Processing instructions |
| Asset path | `banners/launch.jpg` | Original file |

## Starter presets

| Use case | URL token set |
| --- | --- |
| Product thumbnail | `ar_1:1,w_600,c_fill,p_c,q_auto,f_auto` |
| Social open graph | `w_1200,h_630,c_fill,q_auto,f_auto` |
| Large hero | `w_1600,c_fit,q_auto,f_auto` |
| Avatar | `w_240,h_240,c_crop,p_face,br_120,q_auto,f_auto` |

## Example

```text
https://assets.autorender.io/WORKSPACE123/w_1200,h_630,c_fill,q_auto,f_auto/banners/launch.jpg
```
