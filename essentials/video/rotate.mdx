## Overview

Rotate the video by 90, 180, or 270 degrees. Use with width/height as needed for output dimensions. Useful for correcting orientation or fitting layout requirements.

### URL Structure

```bash
https://assets.autorender.io/LOKVTtKVGb/w_200,r_90/doc/skateboarding.mp4
```

<video src="https://assets.autorender.io/LOKVTtKVGb/w_200,r_90/doc/skateboarding.mp4" controls style={{ borderRadius: '0.5rem', maxWidth: '100%', display: 'block', marginLeft: 'auto', marginRight: 'auto', objectFit: 'contain' }} />

## Parameters

| Parameter   | Format  | Description        |
| ----------- | ------- | ------------------ |
| `r_90`      | 90°     | Rotate 90 degrees  |
| `r_180`     | 180°    | Rotate 180 degrees |
| `r_270`     | 270°    | Rotate 270 degrees |
