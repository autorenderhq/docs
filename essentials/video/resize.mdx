## Overview

Resize video by specifying width only, or both width and height. Dimensions are automatically rounded to even values for encoding compatibility. When only width is specified, height is derived to maintain aspect ratio.

Previews below use **fit** (contain): the image or video is shown at the dimensions from the URL (`w_` / `h_`) and scales down only if needed to fit the page; aspect ratio is preserved and nothing is cropped.

### URL Structure

AutoRender video URLs with resize follow this structure:

```bash
https://assets.autorender.io/LOKVTtKVGb/w_500/doc/skateboarding.mp4
```

<video src="https://assets.autorender.io/LOKVTtKVGb/w_500/doc/skateboarding.mp4" controls style={{ borderRadius: '0.5rem', maxWidth: '100%', display: 'block', marginLeft: 'auto', marginRight: 'auto', objectFit: 'contain' }} />

## Parameters

| Parameter | Format      | Description                    |
| --------- | ----------- | ------------------------------ |
| `w_{n}`   | `w_500`     | Width in pixels (1–10000)      |
| `h_{n}`   | `h_500`     | Height in pixels (1–10000)    |

<Note>
  Width and height are automatically rounded to even numbers for H.264 compatibility.
</Note>
