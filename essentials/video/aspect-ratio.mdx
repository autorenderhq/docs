## Overview

Force an aspect ratio (e.g. square 1:1) while resizing. Combine with width or height; the other dimension is calculated automatically to match the ratio.

### URL Structure

```bash
https://assets.autorender.io/LOKVTtKVGb/w_400,ar_1:1/doc/skateboarding.mp4
```

<video src="https://assets.autorender.io/LOKVTtKVGb/w_400,ar_1:1/doc/skateboarding.mp4" controls style={{ borderRadius: '0.5rem', maxWidth: '100%', display: 'block', marginLeft: 'auto', marginRight: 'auto', objectFit: 'contain' }} />

## Parameters

| Parameter   | Format    | Description        |
| ----------- | --------- | ------------------ |
| `ar_{ratio}`| `ar_1:1`  | Aspect ratio       |

Common ratios: `ar_16:9`, `ar_4:3`, `ar_1:1`, `ar_9:16`.

## Example

400px width with 1:1 (square) aspect ratio:

```bash
https://assets.autorender.io/LOKVTtKVGb/w_400,ar_1:1/doc/skateboarding.mp4
```
