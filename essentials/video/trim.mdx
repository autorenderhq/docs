## Overview

Trim the video by specifying start time and duration in seconds. Useful for clips, previews, and reducing file size. Combine with width/height and other transformations as needed.

### URL Structure

```bash
https://assets.autorender.io/LOKVTtKVGb/st_5,d_10,w_500,h_500/doc/skateboarding.mp4
```

<video src="https://assets.autorender.io/LOKVTtKVGb/st_5,d_10,w_500,h_500/doc/skateboarding.mp4" controls style={{ borderRadius: '0.5rem', maxWidth: '100%', display: 'block', marginLeft: 'auto', marginRight: 'auto', objectFit: 'contain' }} />

## Parameters

| Parameter   | Format   | Description              |
| ----------- | -------- | ------------------------ |
| `st_{sec}`  | `st_5`   | Start time in seconds    |
| `d_{sec}`   | `d_10`   | Duration in seconds      |

You can also use `et_{sec}` (end time) in place of duration where supported.

## Example

Start at 5 seconds, keep 10 seconds, resized to 500×500:

```bash
https://assets.autorender.io/LOKVTtKVGb/st_5,d_10,w_500,h_500/doc/skateboarding.mp4
```
