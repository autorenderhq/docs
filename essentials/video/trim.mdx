## Overview

Trim the video by specifying start and end offsets in seconds. Useful for clips, previews, and reducing file size. Combine with width/height and other transformations as needed.

### URL Structure

```bash
https://assets.autorender.io/LOKVTtKVGb/so_5,eo_15,w_500,h_500/doc/skateboarding.mp4
```

<video src="https://assets.autorender.io/LOKVTtKVGb/so_5,eo_15,w_500,h_500/doc/skateboarding.mp4" controls style={{ borderRadius: '0.5rem', maxWidth: '100%', display: 'block', marginLeft: 'auto', marginRight: 'auto', objectFit: 'contain' }} />

## Parameters

| Parameter   | Format   | Description              |
| ----------- | -------- | ------------------------ |
| `so_{sec}`  | `so_5`   | Start offset in seconds  |
| `eo_{sec}`  | `eo_15`  | End offset in seconds    |

## Example

Start at 5 seconds and end at 15 seconds, resized to 500x500:

```bash
https://assets.autorender.io/LOKVTtKVGb/so_5,eo_15,w_500,h_500/doc/skateboarding.mp4
```

</Note>
