## Overview

Mirror the video horizontally, vertically, or both. Flipping is useful for correcting orientation, creating symmetrical previews, or matching layout requirements.

### URL Structure

```bash
https://assets.autorender.io/LOKVTtKVGb/w_400,h_400,flip_h/doc/skateboarding.mp4
```

<video src="https://assets.autorender.io/LOKVTtKVGb/w_400,h_400,flip_h/doc/skateboarding.mp4" controls style={{ borderRadius: '0.5rem', maxWidth: '100%', display: 'block', marginLeft: 'auto', marginRight: 'auto', objectFit: 'contain' }} />

## Parameters

| Parameter | Description                |
| --------- | -------------------------- |
| `flip_h`  | Flip horizontally          |
| `flip_v`  | Flip vertically            |
| `flip_hv` | Flip horizontally and vertically |
