## Overview

Resize the video to fit within the given dimensions and add padding so the full frame is visible. Use `cm_pad_resize` with a background color (e.g. `bg_white`) to avoid letterboxing with black. Ideal for consistent card sizes and fixed aspect ratio layouts.

### URL Structure

```bash
https://assets.autorender.io/LOKVTtKVGb/w_400,h_400,cm_pad_resize,bg_white/doc/skateboarding.mp4
```

## Parameters

| Parameter       | Description                                  |
| --------------- | -------------------------------------------- |
| `cm_pad_resize` | Resize to fit and pad to exact dimensions    |
| `bg_white`      | Padding color (e.g. `bg_white`, `bg_black`, `bg_FF0000`) |

Other background options: `bg_blurred`, `bg_blurred_10`, or hex `bg_FF0000`.

## Example

Fit in 400×400 with white padding:

