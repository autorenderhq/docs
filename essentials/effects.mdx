## Overview

AutoRender provides a comprehensive suite of image enhancement effects that allow you to adjust colors, tones, sharpness, and apply artistic filters. All effects use the `e_` prefix and can be combined with other transformations for professional image processing.

### URL Structure

## Color Enhancements

Adjust the color properties of your images.

### Saturation (`e_saturation`)

Adjusts the intensity/vibrancy of colors.

**Format:** `e_saturation:{level}`
- **Range:** `-100` to `100`
- **Negative values:** Desaturate (move toward grayscale)
- **Positive values:** Increase color intensity

```bash
https://assets.autorender.io/LOKVTtKVGb/e_saturation:30/docs/example/porsche.jpg
```

<img src="https://assets.autorender.io/LOKVTtKVGb/e_saturation:30/docs/example/porsche.jpg" alt="Increased saturation" class="ar-url-image" />

## Effect Processing Order

AutoRender applies effects in an optimized order for best results:

1. Auto Enhance (`e_auto_enhance`)
2. Normalize (`e_normalize`)
3. Exposure & Tonal (`e_exposure`, `e_highlights`, `e_shadows`)
4. Basic Adjustments (`e_brightness`, `e_contrast`, `e_saturation`)
5. Color Adjustments (`e_hue`, `e_temperature`, `e_warmth`, `e_tint`)
6. Gamma (`e_gamma`)
7. Color Space (`e_invert`, `e_grayscale`, `e_blackwhite`)
8. Fade (`e_fade`)
9. Structure (`e_structure`)
10. Sharpening (`e_unsharp_mask`, `e_sharpen`)
11. Vignette (`e_vignette`)
12. Scene Improvements (`e_improve`)

<Note>
  Effects are automatically ordered for optimal results, so you don't need to worry about the order when specifying multiple effects in your URL.
</Note>

## Best Practices

### Effect Combinations

- **Portraits:** `e_auto_enhance`, `e_warmth`, `e_shadows`, `e_structure`, `e_vignette`
- **Landscapes:** `e_normalize`, `e_saturation`, `e_contrast`, `e_structure`, `e_sharpen`
- **Product Photos:** `e_auto_enhance`, `e_brightness`, `e_sharpen`, `e_contrast`
- **Vintage Look:** `e_fade`, `e_temperature`, `e_saturation`, `e_vignette`

### Performance

- Effects processing is optimized for performance
- Multiple effects can be combined efficiently
- Cached results improve performance for repeated requests

### Quality Considerations

- Use moderate values for natural-looking results
- Extreme values may produce artifacts
- Test combinations to find the best balance
