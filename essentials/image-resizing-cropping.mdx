## Overview

AutoRender provides powerful image resizing and cropping capabilities through transformation parameters. You can resize images to specific dimensions, crop to exact sizes, maintain aspect ratios, and control how images are positioned during cropping.

### URL Structure

AutoRender URLs follow this structure:

```
https://assets.autorender.io/LOKVTtKVGb/w_720/docs/example/porsche.jpg
```

<img src="https://assets.autorender.io/LOKVTtKVGb/w_720/docs/example/porsche.jpg" alt="Resized image" class="ar-url-image" />

URL components:
- **Domain**: `https://assets.autorender.io` - AutoRender CDN domain
- **Workspace**: `LOKVTtKVGb` - Your workspace identifier  
- **Transformations**: `w_720` - Image transformation parameters (e.g., width, height, crop modes)
- **Image Path**: `docs/example/porsche.jpg` - Your image path (can include folders)

## Crop Modes

AutoRender supports multiple crop modes that control how images are resized and cropped to fit target dimensions.

### Crop Mode (`c_crop`)

Crops the image to exact dimensions, centering the crop area by default.

```
// Crop to 600x600 square
https://assets.autorender.io/LOKVTtKVGb/w_600,h_600,c_crop/docs/example/porsche.jpg
```


### Fill Mode (`c_fill`)

Resizes the image to cover the entire target area, cropping if necessary to maintain aspect ratio.

```
// Fill 600x600 area (may crop edges)
https://assets.autorender.io/LOKVTtKVGb/w_600,h_600,c_fill/docs/example/porsche.jpg
```

<img src="https://assets.autorender.io/LOKVTtKVGb/w_600,h_600,c_fill/docs/example/porsche.jpg" alt="Resized image" class="ar-url-image" />

### Fit Mode (`c_fit`)

Resizes the image to fit within the target area, preserving the entire image. May add padding if aspect ratios don't match.

```
// Fit within 600x600 (entire image visible)
https://assets.autorender.io/LOKVTtKVGb/w_600,h_600,c_fit/docs/example/porsche.jpg
```

<img src="https://assets.autorender.io/LOKVTtKVGb/w_600,h_600,c_fit/docs/example/porsche.jpg" alt="Fit mode" class="ar-url-image" />

### Scale Mode (`c_scale`)

Scales the image to fit dimensions, potentially distorting the aspect ratio to match exact dimensions.

```
// Scale to exact dimensions (may distort) - 600x400
https://assets.autorender.io/LOKVTtKVGb/w_600,h_400,c_scale/docs/example/porsche.jpg
```

<img src="https://assets.autorender.io/LOKVTtKVGb/w_600,h_400,c_scale/docs/example/porsche.jpg" alt="Resized image" class="ar-url-image" />

<Warning>
  Scale mode may distort images if the target aspect ratio differs from the original. Use `c_fit` or `c_fill` to preserve aspect ratio.
</Warning>

## Best Practices

### Performance Optimization

- Use `c_fit` or `c_fill` instead of `c_scale` to avoid distortion
- Specify only one dimension when possible to let AutoRender calculate the other
- Use aspect ratios for responsive image generation

### Quality Considerations

- AutoRender prevents upscaling by default to maintain quality
- Use appropriate crop modes based on your use case:
  - **`c_crop`**: When you need exact dimensions and can crop content
  - **`c_fill`**: When you need to fill an area completely
  - **`c_fit`**: When you need to show the entire image
  - **`c_scale`**: When exact dimensions are required regardless of distortion
