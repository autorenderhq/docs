## Overview

AutoRender's flip feature allows you to mirror images horizontally, vertically, or both. Flipping is useful for correcting mirrored images, creating symmetrical designs, or adjusting image orientation for different use cases.

## How Flipping Works

Flipping mirrors the image along an axis:
- **Horizontal flip** (`flip_h`): Mirrors the image left-to-right
- **Vertical flip** (`flip_v`): Mirrors the image top-to-bottom
- **Both** (`flip_hv`): Mirrors the image both horizontally and vertically (equivalent to 180° rotation)

## Best Practices

### When to Use Each Flip Type

- **Horizontal flip** (`flip_h`): Most common, used for correcting mirrored images
- **Vertical flip** (`flip_v`): Less common, used for artistic effects or corrections
- **Both** (`flip_hv`): Equivalent to 180° rotation, use when you need both axes flipped

### Performance

- Flip processing is very fast and efficient
- No quality loss when flipping
- Cached results improve performance for repeated requests

### Combining with Other Transformations

- Flip is typically applied after resize and crop operations
- Flip can be combined with rotation for complex orientations
- Consider the final use case when deciding flip direction
