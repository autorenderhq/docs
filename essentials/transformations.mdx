## Transformations

Use one URL pattern for all media:

```text
https://assets.autorender.io/{workspace}/{transformations}/{asset_path}
```

<CardGroup cols={2}>
  <Card title="Image transformations" icon="image" href="/essentials/overview">

  <Card title="Video transformations" icon="film" href="/essentials/video/overview">

  </Card>

<Note>
  Start with `q_auto,f_auto` for images and `thumb_ar` for quick video thumbnails.
</Note>
