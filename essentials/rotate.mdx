## Overview

AutoRender's rotation feature allows you to rotate images by any angle in degrees. Rotation is useful for correcting image orientation, creating artistic effects, or adjusting images that were captured at different angles.

### URL Structure

AutoRender URLs with rotation follow this structure:

```bash

```

## How Rotation Works

Rotation rotates the image around its center point by the specified number of degrees. AutoRender normalizes rotation values to the 0-360 degree range, so you can use any value and it will be converted appropriately.

- **Positive values**: Rotate clockwise
- **Negative values**: Rotate counter-clockwise
- **Values > 360**: Automatically normalized to 0-360 range

## Best Practices

### Rotation Order

- Rotation is typically applied after resize and crop operations
- Consider the final dimensions when rotating, as rotation may change the effective image bounds
- For best results, rotate before applying effects or enhancements

### Performance

- Rotation processing is fast and efficient
- Rotation with transparent backgrounds preserves transparency
- Cached results improve performance for repeated requests

### Quality Considerations

- Rotation uses high-quality interpolation to maintain image quality
- Rotated images may have slightly different dimensions due to the rotation
- For exact dimensions, combine rotation with crop or resize
