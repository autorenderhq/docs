## Overview

AutoRender provides powerful image resizing capabilities that allow you to change image dimensions while maintaining quality and aspect ratios. You can resize images by specifying width, height, or both, and use aspect ratios for responsive image generation.

### URL Structure

AutoRender URLs follow this structure:

```bash
https://assets.autorender.io/LOKVTtKVGb/w_720/doc1/fujicam.jpg
```

<img src="https://assets.autorender.io/LOKVTtKVGb/w_720/doc1/fujicam.jpg" alt="Resized image" style={{ borderRadius: '0.5rem', maxWidth: '100%' }} />

URL components:
- **Domain**: `https://assets.autorender.io` - AutoRender CDN domain
- **Workspace**: `LOKVTtKVGb` - Your workspace identifier  
- **Transformations**: `w_720` - Image transformation parameters
- **Image Path**: `doc/fujicam.jpg` - Your image path (can include folders)

## Width and Height

Resize images by specifying width and/or height dimensions. When only one dimension is specified, AutoRender automatically calculates the other to maintain the original aspect ratio.

### Width Only

Resize by width only - height is calculated automatically to maintain aspect ratio.

```
// Resize to 720px width
https://assets.autorender.io/LOKVTtKVGb/w_720/doc1/gpose.jpg
```

<img src="https://assets.autorender.io/LOKVTtKVGb/w_720/doc1/gpose.jpg" alt="Resized image" style={{ borderRadius: '0.5rem', maxWidth: '100%' }} />

### Height Only

Resize by height only - width is calculated automatically to maintain aspect ratio.

```
// Resize to 540px height
https://assets.autorender.io/LOKVTtKVGb/h_540/doc1/earring.jpg
```

<img src="https://assets.autorender.io/LOKVTtKVGb/h_540/doc1/earring.jpg" alt="Resized image" style={{ borderRadius: '0.5rem', maxWidth: '100%' }} />

### Both Width and Height

Specify both dimensions for exact sizing. When both dimensions are specified, AutoRender uses the appropriate crop mode to achieve the target size.

```
// Resize to exact dimensions (720x480)
https://assets.autorender.io/LOKVTtKVGb/w_720,h_480,c_fill/doc1/paintbrush.jpg
```

<img src="https://assets.autorender.io/LOKVTtKVGb/w_720,h_480,c_fill/doc1/paintbrush.jpg" alt="Resized image" style={{ borderRadius: '0.5rem', maxWidth: '100%' }} />

<Note>
  When only one dimension is specified, AutoRender automatically calculates the other dimension to maintain the original aspect ratio. When both dimensions are specified, you may want to use crop modes (see [Crop](/essentials/crop) page) to control how the image fits the target dimensions.
</Note>

## Relative Sizing

Use decimal values to resize relative to the original image dimensions. This is useful for creating proportional variations of images.

### Relative Width

```
// 50% of original width (maintains aspect ratio)
https://assets.autorender.io/LOKVTtKVGb/w_0.5/doc1/edc.jpg
```

<img src="https://assets.autorender.io/LOKVTtKVGb/w_0.5/doc1/edc.jpg" alt="Resized image" style={{ borderRadius: '0.5rem', maxWidth: '100%' }} />

### Relative Height

```
// 75% of original height
https://assets.autorender.io/LOKVTtKVGb/h_0.75/doc1/earring.jpg
```

<img src="https://assets.autorender.io/LOKVTtKVGb/h_0.75/doc1/earring.jpg" alt="Resized image" style={{ borderRadius: '0.5rem', maxWidth: '100%' }} />

### Common Relative Sizes

```
// 25% of original (thumbnail)
https://assets.autorender.io/LOKVTtKVGb/w_0.25/doc1/edc.jpg

// 50% of original (medium)
https://assets.autorender.io/LOKVTtKVGb/w_0.5/doc1/edc.jpg

// 75% of original (large)
https://assets.autorender.io/LOKVTtKVGb/w_0.75/doc1/edc.jpg

// 150% of original (enlarged - may reduce quality)
https://assets.autorender.io/LOKVTtKVGb/w_1.5/doc1/edc.jpg
```

<Warning>
  Enlarging images beyond their original size (values > 1.0) may reduce image quality. AutoRender prevents upscaling by default to maintain quality, but you can override this behavior when needed.
</Warning>

### Custom Aspect Ratios

You can use any aspect ratio by specifying the ratio in the format `width:height`:

```
// 3:2 aspect ratio
https://assets.autorender.io/LOKVTtKVGb/ar_3:2,w_900,c_fill/doc1/fujicam.jpg

// 5:4 aspect ratio
https://assets.autorender.io/LOKVTtKVGb/ar_5:4,w_1000,c_fill/doc1/paintbrush.jpg

// 9:16 portrait aspect ratio
https://assets.autorender.io/LOKVTtKVGb/ar_9:16,h_1080,c_fill/doc1/gpose.jpg
```

<Note>
  When using aspect ratio, specify either width or height. AutoRender calculates the other dimension automatically. Aspect ratios work best when combined with crop modes (see [Crop](/essentials/crop) page) to control how the image is cropped to fit the ratio.
</Note>

Use AutoRender URLs with HTML `srcset` for automatic responsive image selection:

```html
<img 
  src="https://assets.autorender.io/LOKVTtKVGb/w_400/doc1/rotjump.jpg"
  srcset="
    https://assets.autorender.io/LOKVTtKVGb/w_400/doc1/rotjump.jpg 400w,
    https://assets.autorender.io/LOKVTtKVGb/w_800/doc1/rotjump.jpg 800w,

    https://assets.autorender.io/LOKVTtKVGb/w_1920/doc1/rotjump.jpg 1920w
  "
  sizes="(max-width: 600px) 400px, (max-width: 1200px) 800px, 1200px"
  alt="Responsive image"
/>
```

## Best Practices

### Performance Optimization

- **Use bucket sizes**: Specify widths that match AutoRender's width buckets (320, 480, 720, 1080, 1440, 1920) for optimal caching
- **Specify one dimension**: When possible, specify only width or height to let AutoRender calculate the other automatically
- **Use aspect ratios**: For responsive images, use aspect ratios to maintain consistent proportions

### Quality Considerations

- **Avoid upscaling**: AutoRender prevents upscaling by default to maintain quality
- **Use appropriate sizes**: Request images at the size you need rather than resizing in CSS
- **Consider format**: Combine with format conversion (WebP, AVIF) for better compression
