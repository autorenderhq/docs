## Overview

AutoRender provides powerful image resizing capabilities that allow you to change image dimensions while maintaining quality and aspect ratios. You can resize images by specifying width, height, or both, and use aspect ratios for responsive image generation.

Previews in this guide use a **maximum of 360px** width/height so you can see results at a glance. They are shown at **real output dimensions** (fit/contain) and scale down on narrow viewports.

### URL Structure

AutoRender URLs follow this structure:

```bash
https://assets.autorender.io/LOKVTtKVGb/w_360/doc1/fujicam.jpg
```

<img src="https://assets.autorender.io/LOKVTtKVGb/w_360/doc1/fujicam.jpg" alt="Resized image" style={{ width: 360, maxWidth: '100%', height: 'auto', borderRadius: '0.5rem', display: 'block', marginLeft: 'auto', marginRight: 'auto', objectFit: 'contain' }} />

URL components:
- **Domain**: `https://assets.autorender.io` - AutoRender CDN domain
- **Workspace**: `LOKVTtKVGb` - Your workspace identifier  
- **Transformations**: `w_360` - Image transformation parameters
- **Image Path**: `doc/fujicam.jpg` - Your image path (can include folders)

### Custom Aspect Ratios

You can use any aspect ratio by specifying the ratio in the format `width:height`:

```
// 3:2 at 360px width
https://assets.autorender.io/LOKVTtKVGb/ar_3:2,w_360,c_fill/doc1/fujicam.jpg

// 5:4 at 360px width
https://assets.autorender.io/LOKVTtKVGb/ar_5:4,w_360,c_fill/doc1/paintbrush.jpg

// 9:16 portrait at 360px height
https://assets.autorender.io/LOKVTtKVGb/ar_9:16,h_360,c_fill/doc1/gpose.jpg
```

<Note>
  When using aspect ratio, specify either width or height. AutoRender calculates the other dimension automatically. Aspect ratios work best when combined with crop modes (see [Crop](/essentials/crop) page) to control how the image is cropped to fit the ratio.
</Note>

## Best Practices

### Performance Optimization

- **Use bucket sizes**: Specify widths that match AutoRender's width buckets (320, 480, 720, 1080, 1440, 1920) for optimal caching
- **Specify one dimension**: When possible, specify only width or height to let AutoRender calculate the other automatically
- **Use aspect ratios**: For responsive images, use aspect ratios to maintain consistent proportions

### Quality Considerations

- **Avoid upscaling**: AutoRender prevents upscaling by default to maintain quality
- **Use appropriate sizes**: Request images at the size you need rather than resizing in CSS
- **Consider format**: Combine with format conversion (WebP, AVIF) for better compression
