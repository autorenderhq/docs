## Overview

The grayscale effect converts images to grayscale (black and white), removing all color information. This creates classic, timeless black and white images.

### URL Structure

AutoRender URLs with grayscale follow this structure:

```bash
https://assets.autorender.io/LOKVTtKVGb/e_grayscale/docs/example/porsche.jpg
```

## Grayscale (`e_grayscale`)

Converts image to grayscale.

**Format:** `e_grayscale`

```bash
https://assets.autorender.io/LOKVTtKVGb/e_grayscale/docs/example/porsche.jpg
```

<img src="https://assets.autorender.io/LOKVTtKVGb/e_grayscale/docs/example/porsche.jpg" alt="Grayscale image" class="ar-url-image" />

## Use Cases

- **Classic photography:** Create timeless black and white images
- **Artistic effects:** Emphasize composition and contrast
- **Product photography:** Create elegant, minimalist product images
- **Portrait photography:** Create dramatic, classic portraits

## Combining with Other Effects

Grayscale works well with contrast and structure:

```bash
https://assets.autorender.io/LOKVTtKVGb/e_grayscale,e_contrast:40,e_structure:50/docs/example/porsche.jpg
```

<img src="https://assets.autorender.io/LOKVTtKVGb/e_grayscale,e_contrast:40,e_structure:50/docs/example/porsche.jpg" alt="Grayscale with contrast and structure" class="ar-url-image" />

## Related Effects

- **Black & White (`e_blackwhite`)**: Provides threshold control for black/white conversion
- **Invert (`e_invert`)**: Creates negative/inverted images

## Best Practices

- Combine with contrast for dramatic black and white images
- Use structure to enhance detail in grayscale images
- Grayscale emphasizes composition and lighting
- Test different combinations for artistic effects
