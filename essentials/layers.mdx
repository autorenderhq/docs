## Overview

AutoRender Layers allow you to composite text and images on top of or beneath your base image. Layers enable you to add watermarks, logos, text overlays, badges, and complex multi-image compositions without requiring external image editing software.

### URL Structure

AutoRender URLs with layers follow this structure:

```bash
https://assets.autorender.io/LOKVTtKVGb/l_text:arial_24_bold_center:Hello%20World,tp_north-east,fl_layer_apply,w_400,h_400/docs/example/porsche.jpg
```

## Text Layers

Add text overlays to your images with full control over font, size, color, position, and styling.

### Basic Text Layer

**Format:** `l_text:{fontFamily}_{fontSize}_{style}_{align}:{text}`

Simple text overlay:

```bash
https://assets.autorender.io/LOKVTtKVGb/l_text:arial_24_bold_center:Hello%20World,tp_north-east,fl_layer_apply,w_400,h_400/docs/example/porsche.jpg
```

**Parameters:**
- **Font Family**: Font name (e.g., `arial`, `helvetica`, `times`)
- **Font Size**: Size in pixels (e.g., `24`, `48`)
- **Style**: `bold`, `italic`, `underline` (optional, can combine)
- **Align**: `left`, `center`, `right`
- **Text**: URL-encoded text content

### Text Color

Set text color using `co_` prefix:

**Format:** `co_{hexColor}` or `co_rgb:{hexColor}`

Red text:

```bash
https://assets.autorender.io/LOKVTtKVGb/l_text:arial_32_bold_center:Sale,co_FF0000,fl_layer_apply,w_400,h_400/docs/example/porsche.jpg
```

White text:

```bash
https://assets.autorender.io/LOKVTtKVGb/l_text:arial_48_bold_center:NEW,co_FFFFFF,fl_layer_apply,w_400,h_400/docs/example/porsche.jpg
```

### Text Position

Position text using X and Y coordinates:

**Format:** `x_{value},y_{value}`

Position text at specific coordinates:

```bash
https://assets.autorender.io/LOKVTtKVGb/l_text:arial_24_bold_center:Watermark,x_100,y_50,fl_layer_apply,w_400,h_400/docs/example/porsche.jpg
```

### Text Placement

Use automatic placement instead of manual coordinates:

**Format:** `tp_{placement}`

**Placement options:**
- `tp_north` / `tp_n` - Top center
- `tp_south` / `tp_s` - Bottom center
- `tp_east` / `tp_e` - Right center
- `tp_west` / `tp_w` - Left center
- `tp_center` / `tp_c` - Center
- `tp_north-west` / `tp_nw` - Top-left
- `tp_north-east` / `tp_ne` - Top-right
- `tp_south-west` / `tp_sw` - Bottom-left
- `tp_south-east` / `tp_se` - Bottom-right

```bash
https://assets.autorender.io/LOKVTtKVGb/l_text:arial_32_bold_center:Title,tp_north,co_FFFFFF,fl_layer_apply,w_400,h_400/docs/example/porsche.jpg
```

## Image Layers

Add image overlays to your base image. For image layers, keep it simple: use the layer source plus layer dimensions (`lw_`, `lh_`) and apply flag.

### Basic Image Layer

**Format:** `l_image:{imagePath}`

Add image overlay:

```bash
https://assets.autorender.io/LOKVTtKVGb/l_image:logo.svg,fl_layer_apply,w_400,h_400/docs/example/porsche.jpg
```

</Note>

## Best Practices

### Layer Order

- Text layers are applied first
- Image layers are applied after text layers
- Multiple layers are processed in sequence

### Performance

- Layer processing adds overhead
- Cache layered images when possible
- Use appropriate layer sizes to avoid unnecessary processing
- Combine multiple layers efficiently

### Layer Sizing

- Resize layers appropriately for the base image
- Use both `lw_` and `lh_` for predictable layer output
- Consider the final output size when sizing layers
