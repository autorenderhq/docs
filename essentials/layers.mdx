## Overview

AutoRender Layers allow you to composite text and images on top of or beneath your base image. Layers enable you to add watermarks, logos, text overlays, badges, and complex multi-image compositions without requiring external image editing software.

### URL Structure

AutoRender URLs with layers follow this structure:

```bash
https://assets.autorender.io/LOKVTtKVGb/l_text:arial_24_bold_center:Hello%20World,fl_layer_apply/docs/example/porsche.jpg
```

## Text Layers

Add text overlays to your images with full control over font, size, color, position, and styling.

### Basic Text Layer

**Format:** `l_text:{fontFamily}_{fontSize}_{style}_{align}:{text}`

Simple text overlay:

```bash
https://assets.autorender.io/LOKVTtKVGb/l_text:arial_24_bold_center:Hello%20World,fl_layer_apply/docs/example/porsche.jpg
```

**Parameters:**
- **Font Family**: Font name (e.g., `arial`, `helvetica`, `times`)
- **Font Size**: Size in pixels (e.g., `24`, `48`)
- **Style**: `bold`, `italic`, `underline` (optional, can combine)
- **Align**: `left`, `center`, `right`
- **Text**: URL-encoded text content

### Text Color

Set text color using `co_` prefix:

**Format:** `co_{hexColor}` or `co_rgb:{hexColor}`

### Text Placement

Use automatic placement instead of manual coordinates:

**Format:** `tp_{placement}`

**Placement options:**
- `tp_north` / `tp_n` - Top center
- `tp_south` / `tp_s` - Bottom center
- `tp_east` / `tp_e` - Right center
- `tp_west` / `tp_w` - Left center
- `tp_center` / `tp_c` - Center
- `tp_north-west` / `tp_nw` - Top-left
- `tp_north-east` / `tp_ne` - Top-right
- `tp_south-west` / `tp_sw` - Bottom-left
- `tp_south-east` / `tp_se` - Bottom-right


## Image Layers

Add image overlays or underlays to composite multiple images together.

### Basic Image Layer

**Format:** `l_image:{imagePath}`

Add image overlay:

```bash
https://assets.autorender.io/LOKVTtKVGb/l_image:logo.png,fl_layer_apply/docs/example/porsche.jpg
```

### Layer Type

Specify whether the layer is an overlay or underlay:

**Format:** `l_type:{type}`

**Types:**
- `overlay` - Layer appears on top (default)
- `underlay` - Layer appears beneath the base image

Underlay image:

```bash
https://assets.autorender.io/LOKVTtKVGb/l_image:background.jpg,l_type:underlay,fl_layer_apply/docs/example/porsche.jpg
```

Overlay image (default):

```bash
https://assets.autorender.io/LOKVTtKVGb/l_image:watermark.png,l_type:overlay,fl_layer_apply/docs/example/porsche.jpg
```

### Layer Dimensions

Resize the layer image:

**Format:** `lw_{width},lh_{height}`

Resize layer to 200x100:

```bash
https://assets.autorender.io/LOKVTtKVGb/l_image:logo.png,lw_200,lh_100,fl_layer_apply/docs/example/porsche.jpg
```

Apply effects to image layers:

**Format:** `le_{effect}`

**Available layer effects:**
- `le_saturation:{level}` - Color saturation
- `le_contrast:{level}` - Image contrast
- `le_brightness:{level}` - Image brightness
- `le_sharpen:{strength}` - Sharpening
- `le_grayscale` - Grayscale conversion

## Best Practices

### Layer Order

- Text layers are applied first
- Image layers are applied after text layers
- Underlays are processed before overlays
- Multiple layers are processed in sequence

### Performance

- Layer processing adds overhead
- Cache layered images when possible
- Use appropriate layer sizes to avoid unnecessary processing
- Combine multiple layers efficiently

### Positioning

- Use placement (`tp_`, `ip_`) for automatic positioning
- Use coordinates (`x_`, `y_`) for precise control
- Consider image dimensions when positioning layers

### Layer Sizing

- Resize layers appropriately for the base image
- Use aspect ratios to maintain proportions
- Consider the final output size when sizing layers
