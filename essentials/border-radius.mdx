## How Border Radius Works

Border radius rounds the corners of an image by the specified number of pixels. The radius value determines how rounded the corners will be:
- **Small values** (5-20px): Subtle rounding
- **Medium values** (20-50px): Moderate rounding
- **Large values** (50-100px): Very rounded corners
- **Very large values** (≥ half the smallest dimension): Creates circular/oval shapes

## Border Radius Guidelines

### Recommended Values

| Use Case | Border Radius | Example |
|----------|---------------|---------|
| Subtle rounding | 5-15px | `br_10` |
| Card layouts | 15-25px | `br_20` |
| Modern UI | 20-40px | `br_30` |
| Very rounded | 40-60px | `br_50` |
| Circular (square images) | ≥ half dimension | `br_150` for 300x300 |

### Best Practices

- **Small images**: Use smaller radius values (5-20px)
- **Large images**: Can use larger radius values (20-50px)
- **Square images**: Use radius ≥ half dimension for circular effect
- **Rectangular images**: Radius creates rounded rectangle effect

## Best Practices

### Performance

- Border radius processing is fast and efficient
- No quality loss when applying border radius
- Cached results improve performance for repeated requests

### Design Considerations

- Match border radius to your design system
- Consistent radius values create visual harmony
- Consider the image content when choosing radius size
- Very large radius values may crop important image content

### Combining with Other Transformations

- Border radius is typically applied after resize and crop
- Works well with all crop modes
- Can be combined with rotation and flip for creative effects
