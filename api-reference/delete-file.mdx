<RequestExample>

```bash cURL
curl --request DELETE \
  --url 'https://upload.autorender.io/api/v1/files/2338056701' \
  --header 'Authorization: Bearer YOUR_API_KEY'
```

</RequestExample>

<ResponseExample>

```json 200
{
  "message": "file deleted successfully"
}
```

```json 404
{
  "error": "File not found"
}
```

```json 401
{
  "success": false,
  "error": "Unauthorized",
  "message": "Invalid or missing API key"
}
```

</ResponseExample>

## Overview

Deletes the file identified by **`fileNo`**. No request body is required.

<Warning>
  Deletion is permanent (storage and database). This cannot be undone.
</Warning>

## Path parameters

| Parameter | Description |
| --------- | ----------- |
| `fileNo` | File number to delete |
