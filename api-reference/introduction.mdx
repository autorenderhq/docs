## Base URLs

- **Upload API:** `https://upload.autorender.io`
- **Delivery CDN:** `https://assets.autorender.io`

## Authentication

Send your API key as a bearer token on every upload and management request.

```bash
Authorization: Bearer YOUR_API_KEY
```

## Typical backend flow

1. Upload file to `/api/v1/uploads`.
2. Store returned `file_no`, `path`, and `url` in your database.
3. Deliver transformed variants through CDN URL tokens.
4. Use list/detail endpoints for moderation, catalog, and reporting tools.
