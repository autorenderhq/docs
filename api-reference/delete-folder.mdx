## Overview

Removes the folder identified by **`folderNo`**. No request body is required.

<Warning>
  Ensure the folder is empty or that your product rules allow deletion; this may fail if the folder still contains files.
</Warning>

## Path parameters

| Parameter | Description |
| --------- | ----------- |
| `folderNo` | Folder number to delete |
