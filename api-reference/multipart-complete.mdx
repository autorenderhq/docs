## Overview

Completes the multipart session after every part has been uploaded with **PUT** to the presigned URLs. The file is assembled and registered in your workspace.

## Authentication

`Authorization: Bearer YOUR_API_KEY`

## Workflow

This is **step 3 of 3**.

1. [**Multipart start**](/api-reference/multipart-start)
2. [**Upload parts**](/api-reference/multipart-upload-parts)
3. **Multipart complete** (this endpoint) — body: `{ "session_id": "<from start>" }`

## Request body

| Field | Type | Required | Description |
| ----- | ---- | -------- | ----------- |
| `session_id` | string | Yes | `session_id` from the start response |
