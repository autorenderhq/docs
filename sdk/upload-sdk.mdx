## What this feature covers

- File picker and upload trigger
- Progress + retry handling
- Response parsing (`file_no`, `path`, `url`)

## Security model

- Generate short-lived signed/scoped upload credentials from backend.
- Never expose long-lived admin API keys in frontend code.
